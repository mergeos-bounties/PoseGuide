# Standing pose templates

Each JSON file is a **standing** photography pose template:

| Field | Meaning |
| --- | --- |
| `id` | Stable slug |
| `tags` | Scene tags this pose fits |
| `tips` | Coach text for the subject |
| `camera_cues` | Photographer framing notes |
| `joints` | Normalized skeleton keypoints (0–1 image space) |

Synthetic scaffold data only — expand via bounties with consented / licensed references.
